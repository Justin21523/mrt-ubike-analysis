About / Methods
Data sources
  • TDX · Metro/Bike open data (OAuth2), collected into Bronze → Silver
  • Open‑Meteo · Hourly weather (no API key), stored as data/external/weather_hourly.csv
Key definitions
  • available · nearby bike availability aggregated to each MRT station
  • rent_proxy / return_proxy · proxy demand signals derived from bike snapshots
Limitations
  • Weather is a city point estimate (not per-station microclimate).
  • Proxy signals are directional indicators; validate before policy decisions.
